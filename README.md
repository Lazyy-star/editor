# Cyber Zen Writer

在线体验：
[https://lazyy-star.github.io/editor/](https://lazyy-star.github.io/editor/)

一个纯前端的沉浸式文字编辑器，主打雨雪氛围、极简写作和本地隐私保存。

## Features

- `Rain / Snow` 双场景切换
- 沉浸式毛玻璃写作区
- 隐藏式 `VibeMixer` 氛围控制台
- 可调雨量/雪量、速度、模糊、白噪音、音乐融合度
- `Zen Timer` 心流计时器
- LocalStorage 实时暂存
- 导出 `.txt`
- 纯前端运行，无需后端

## Run Locally

直接打开 `index.html` 即可。


## Notes

- 写作内容默认保存在浏览器本地存储中
- 导出文件格式当前为 `.txt`
- 建议使用现代浏览器获得更完整的视觉效果
